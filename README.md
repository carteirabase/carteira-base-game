# Carteira Base — Batalha 8-bit

Este repositório contém um mini-jogo HTML jogável no navegador.

## Como publicar no GitHub Pages
1. Crie um repositório no GitHub.
2. Faça upload destes arquivos (pelo celular mesmo).
3. Vá em **Settings → Pages** e escolha:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /(root)
4. Salve e abra o link que o GitHub gerar.

## Monetização
- **Apoio/PIX**: edite o `SUPPORT_URL` dentro do `index.html`.
- **AdSense**: depois de aprovado, preencha `ADSENSE_CLIENT` e `ADSENSE_SLOT_*` no `index.html` e coloque sua linha em `ads.txt`.
